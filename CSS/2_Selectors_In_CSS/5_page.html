Selectors in HTML

Element Selector

The element selectors select HTML element based on the element name

ID Selectors

The id selectors use the id attribute of an HTML element to select a specific element. The id element is unique within a page, so the id selector is used to select one specific element

Class Selectors

The class selector selects the HTML element with a specific class attribute. To select elements with a specific class, write a period(.) character, followed by the class name
